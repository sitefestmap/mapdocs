There are some examples in the docs of how the map site might be coded in a JavaScript framework. Solid JS is used but there are also other frameworks that have Mapbox adapters such as Svelte and React.

```bash
# https://github.com/birkskyum/solid-maplibre
# https://github.com/GIShub4/solid-map-gl
npm i mapbox-gl solid-map-gl

# Solid Start
pnpm create solid && pnpm i && \
pnpm add mapbox-gl solid-map-gl && \
pnpm dev
```
